\doxysubsection{Detailed Description}
AJAX pro filtrování a paginaci produktů. Tento soubor zpracovává GET parametry z frontend filtru (main.\+js), sestavuje dynamický SQL query na základě filtrů (search, size, price range, season, page), vrací HTML s produkty a paginací. Používá se pro dynamické načítání produktů bez reloadu stránky. Výstup je přímo HTML pro vložení do \#products elementu.
