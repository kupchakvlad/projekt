\doxysubsection{Detailed Description}
Backend skript pro zpracování přidání nového produktu. Tento soubor zpracovává POST data a nahrané soubory z formuláře \doxylink{add__product_8php}{add\+\_\+product.\+php}. Provádí validaci vstupů (fotky, název, cena), nahrává obrázky na server, ukládá jejich cesty do databáze a vkládá záznam o produktu do tabulky {\ttfamily products}. Při chybách ukládá errory a data do session a přesměruje zpět na formulář. Při úspěchu přesměruje na hlavní stránku (\doxylink{main_8php}{main.\+php}).
