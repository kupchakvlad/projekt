Detailed Description

Stránka uživatelského profilu / účtu (My Account). Tento soubor je přístupný pouze přihlášeným uživatelům. Načítá aktuální jméno a email uživatele z databáze, zobrazuje je, umožňuje jejich úpravu a změnu hesla přes formulář (odesílá na account_back.php). Podporuje tmavý režim z cookie, obsahuje tlačítka pro návrat, přepnutí režimu a odhlášení.
