Detailed Description

Backend skript pro přepnutí administrátorského oprávnění uživatele.

Tento soubor je přístupný pouze přihlášeným administrátorům. Na základě ID z GET parametru načte aktuální admin status uživatele, přepne jej (0 → 1 nebo 1 → 0) a uloží změnu do databáze pomocí prepared statements. Zabraňuje přepnutí vlastního účtu a neplatnému ID. Po úspěšné (nebo neúspěšné) operaci přesměruje zpět na admin.php.
